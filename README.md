1. Pergi ke folder `Filtering` dengan command `cd Filtering`, setelah di folder Filtering, masuk lagi ke folder `backend` dengan command `cd backend`
2. Jalankan command `pip install -r requirements.txt` di terminal untuk menginstall library yang dibutuhkan
3. Lalu jalankan command `python -m uvicorn main:app --reload` di terminal untuk menjalankan website
4. Klik link localhost yang akan ter-print pada terminal untuk mengakses websitenya
5. Website sudah bisa diakses ketika ada pesan `Application startup complete` pada terminal.
