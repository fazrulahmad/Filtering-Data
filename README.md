1. Jalankan command `pip install -r requirements.txt` untuk menginstall library yang dibutuhkan
2. Lalu jalankan command `python -m uvicorn main:app --reload` untuk menjalankan website
3. Klik link localhost yang akan ter-print pada terminal untuk mengakses websitenya
4. Website sudah bisa diakses ketika ada pesan `Application startup complete` pada terminal.
